BBS SAMPLE

  • PROFILE FORM
    - プロフィール情報を登録するフォームページです。
  • PROFILE LIST
    - PROFILE FORMで登録したプロフィール一覧・各プロフィールページです。
  • BBS TEST
    - 掲示板ページです。
  • BROADCAST
    - 超簡易版レス遅延報告掲示板です。
    - この掲示板に登録した遅延情報は、最新2日分が掲示板ページのトップに掲載されます。
  • ADMIN
    - 管理者用ページです。
    - すべてのプロフィール情報の修正機能があります。
